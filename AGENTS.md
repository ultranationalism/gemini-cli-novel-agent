# Gemini CLI Novel - 端到端小说创作系统

## 项目简介

这是一个基于 MCP (Model Context Protocol) 工具链的端到端小说创作系统。系统通过集成多种工具，实现从构思、写作到修正的完整创作流程。

## 核心能力

- **自动化创作流程**：支持从大纲生成到章节写作的全流程自动化
- **多维度记忆系统**：通过 memory、openmemory、milvus 实现设定、剧情、灵感的立体化管理
- **智能辅助工具**：集成逻辑推演和现实细节检索能力
- **自主创作模式**：支持基于灵感检索的完全自主创作

## 技术架构

### 记忆系统
- **memory**: 静态设定存储
- **openmemory**: 时间线日志管理
- **milvus**: 向量化灵感检索库

### 辅助工具
- **sequential-thinking**: 深层逻辑推演
- **tavily**: 现实细节联网检索

## 项目结构

禁止在项目主目录下使用ls，agent仅被允许在novel文件夹中工作

```
novel/
  ├── [book_name]/
  │   ├── plan.md          # 总控文件
  │   ├── 正文/             # 章节正文
  │   └── summary/         # 章节摘要
```

## 质量标准

- 单章字数：≥ 10,000 字
- 作品规模：≥ 100,000 字
- 创作方式：分步构思，分段完成
